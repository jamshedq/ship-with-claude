# Raw Inbox

**Drop anything in here.** Links. Screenshots. Half-thoughts. Overheard quotes. Random observations. Article URLs you want to react to. Tweets that hit. Build moments. No formatting rules. No structure required.

The Sunday sweep reads this file alongside `inbox.md` and figures out what could become a post — even from messy fragments.

---

## How it works

**You during the week:** dump raw stuff below the divider. Date stamps are nice but not required. No bullets needed. Whatever's fastest to capture.

**The Sunday sweep:** reads each item and decides:

| Marker | Meaning |
|--------|---------|
| (no marker) | New / unprocessed — Claude will evaluate this Sunday |
| `[drafted: slug]` | Claude turned it into a post — see `drafts/<slug>.md` |
| `[promoted]` | Claude moved it into `inbox.md` as a structured brief (you can review there) |
| `[skipped: reason]` | Claude looked at it but couldn't make a post out of it |
| `[ask: question]` | Claude needs more context from you before it can use this |

You don't need to clean this file out — Claude marks items as it processes. You can delete `[drafted]` and `[skipped]` items whenever you want a tidier file.

## What kinds of things to drop here

- A link + a one-line take ("interesting argument on agent eval — disagree with point 3")
- A screenshot path + context ("see this Cowork session — sub-agent ran 4 in parallel, 8 sec total")
- A number you heard or measured ("our cost dropped 73% switching to Haiku for triage")
- A half-formed observation ("everyone's building agents but nobody's measuring drift")
- An article URL you want to react to
- A quote from a meeting / podcast / Twitter thread
- A tweet you wish you'd written
- A question you got asked twice this week
- A bug or surprise from your own work
- Anything that made you think "huh, that's a post"
