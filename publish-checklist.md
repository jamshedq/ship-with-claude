# Publish Checklist

60-second pre-flight before you hit Post. Every draft from `drafts/` runs through this. Designed to catch the small misses that tank a good post.

## Pre-publish (60 seconds)

### Hook & body

- [ ] **Hook is ≤49 characters** (LinkedIn truncates above this in the feed)
- [ ] **First two lines do all the work** — pull the reader past "see more" without needing the rest
- [ ] **Caption is 1,200–1,800 characters** (sweet spot for dwell time and reach)
- [ ] **Line break every 1–2 sentences** — no walls of text
- [ ] **One idea, one CTA** — if there are two CTAs, kill one
- [ ] **No banned phrases** (cross-check with `voice-samples.md`)
- [ ] **At least one substance marker present** — artifact, number, failure mode, comparison rule, or named pattern

### Carousel (if applicable)

- [ ] **Open the populated `.html` in `drafts/` in Chrome** (auto-generated alongside the `.md` by the loop)
- [ ] **Cmd+P → Save as PDF** — Layout: Portrait, Pages: All, Margins: None, Background graphics: ON
- [ ] **Page count is driven by substance** — 2 pages for short tips, 3 standard, 4–5+ for deep dives. Never compress or pad to hit a number.
- [ ] **Page 1 carries the hook** in the title block
- [ ] **Page N has the full orange footer bar** with B&W headshot circle (every other page uses the small footer)
- [ ] **Tagline reads "Hands-on with AI. Less hype, more shipping."** in every footer
- [ ] **Uploaded as a Document on LinkedIn** (not as separate images)

### Links

- [ ] **No links in the post body** (cuts reach 40–60%)
- [ ] **Any link goes in the first comment**, posted by you within 30 seconds of publishing
- [ ] **First comment is drafted and ready to paste** before hitting Post

### Hashtags

- [ ] **3–5 hashtags total**
- [ ] **Mix: 1 large (#AI / #SaaS) + 2 mid + 1 niche** (per `playbook.md`)
- [ ] **Niche tag is specific** (#ClaudeCode / #MCP / #ClaudeCowork — not generic)

### Images & alt text

- [ ] **Every image has alt text** — affects accessibility AND a small reach signal
- [ ] **Image dimensions correct** (1080×1350 for carousels, 1200×627 for single-image link previews)

### Tagging

- [ ] **Tagged people are actually relevant** — random tagging is demoted
- [ ] **Tagged people are likely to engage** in the first hour

### Timing

- [ ] **Posting at the scheduled time** for the day (per `playbook.md` weekly cadence)
- [ ] **Not posting on top of a major news event** that will drown the post (move it ±2 hours if needed)

## Hit Post

## Post-publish (within 60 minutes)

- [ ] **Drop the first comment** with link / source / "P.S." within 30 seconds
- [ ] **Reply to every comment** as it comes in
- [ ] **DM 3 people** who'd genuinely care, with the post link and one honest sentence
- [ ] **Do NOT edit the post** in the first hour (early edits demote reach)
- [ ] **Add a Day-0 row to `performance-log.md`** (just the row stub — metrics come at D2 and D7)

## Day 2

- [ ] Open the post in LinkedIn analytics
- [ ] Fill **D2 reach + engagement + comments** in `performance-log.md`
- [ ] If already at 2x baseline reach: write a follow-up brief into `inbox.md` for next week

## Day 7

- [ ] Fill **D7 reach + engagement rate + saves + profile views + followers gained** in `performance-log.md`
- [ ] Write the **learning** column — one sentence
- [ ] If 5x+ baseline: pitch a podcast or write a long-form follow-up
- [ ] If <50% baseline: nothing yet — single data point, not a pattern

## Quick reject — kill the draft if any of these are true

- [ ] You can't name what's specific about it in one sentence
- [ ] The hook could appear above 50% of LinkedIn AI posts this week
- [ ] You'd be embarrassed if a senior practitioner in your audience read it
- [ ] You used "AI", "transform", "unlock", or "game-changer" anywhere
- [ ] You're publishing it because you "have to post today" — better to skip a day

## Speedrun version (when you trust the draft)

If you've used the loop for 2+ weeks and the draft already looks clean:

1. Hook ≤49 chars ✓
2. No body links ✓
3. 3–5 hashtags ✓
4. First comment ready ✓
5. Post → first comment → reply mode

That's it. The detailed checklist above is for the first month while patterns are still forming.
